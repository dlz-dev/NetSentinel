# NetSentinel 🛡️

Real-time network intrusion detection pipeline built with Apache Spark.

## Stack
- Apache Spark 3.5.5
- PySpark 3.5.1
- Docker
- Jupyter Notebook

## Setup
```bash
docker-compose up
```
Then open Jupyter at `http://localhost:8889`

## Dataset
BCCC-CIC-IDS-2017 (MIT, 2024) — 2GB of labeled network traffic.
